# Automated-Testing-for-nopCommerce-website
Automated testing for nopCommerce demo store website

Automated loginPage for website : https://demo.nopcommerce.com/

- Use Selenium WebDriver + Java pour l'automatisation UI

- Use TestNG Framework and Page Object Model (POM)

- Use Jenkins CI/CD pour l'orchestration des tests

  https://github.com/mohammedtoumi007/Automated-Testing-for-nopCommerce-website/issues/1#issue-3676556868
  
- Use Allure Reports pour rapport de test visuels et détaillés
